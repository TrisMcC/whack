# This is a simple Whack-a-Mole game.

"Moles" (identified by yellow color) appear and the player is given 1 second to click on the mole. The player wins after wacking 5 moles.

This project has no dependencies and requires no server. It demonstrates basic HTML5, CSS3, and ES6.

This has been tested in Firefox 53.0, Chrome 58.0.3029.81, and Microsoft Edge 38.14393.1066.0.
